# Tiny Tetris

## About

A Tetris implementation that "fits on a business card" (23 lines x 80 characters)

<img src="https://i.imgur.com/Tb5VCwb.gif" alt="Game in progress" width="40%">

## Installation

```bash
git clone https://github.com/nickmpaz/tiny-tetris
cd tiny-tetris && python3 tiny-tetris.py
```
## Controls

- up: rotate
- left: move left
- right: move right
- down: move down
- ctrl+c: quit
